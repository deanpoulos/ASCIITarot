# ASCII-Tarot
## Description
An aesthetic command-line tool for randomised tarot readings and (eventually)
combinations and integrations of various divination tools.

## Installation
``` console
$ git clone https://github.com/deanpoulos/ASCIITarot.git
$ cd ASCIITarot
$ pip install .
```
